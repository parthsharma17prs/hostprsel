# Aura Living — Premium Student Housing

A high-performance, visually stunning React web application for premium hostel and student residency management. Designed with a focus on **luxury aesthetics**, **ultra-smooth user experience**, and **interactive storytelling**.

## ✨ Premium Features

- **🌊 Ultra-Smooth Scrolling**: Powered by Lenis with custom lerp and easing for a "buttery" feel.
- **🖼️ Awwwards-Inspired Interactions**:
  - **Magnetic Pull**: Dynamic gravity effect on all interactive buttons, logos, and social icons.
  - **3D Perspective Tilt**: Interactive image hover effects that react to cursor position.
  - **Dual-Layer Cursor**: A fast pinpoint dot with a lagging elastic ring for tactile feedback.
- **🎬 Cinematic Experience**:
  - **Split-Screen Preloader**: A sophisticated multi-panel entrance with animated counters.
  - **Staggered Reveals**: GSAP-driven text and element entrance animations.
  - **Ambient Glows**: Subtle floating background elements for depth and premium atmosphere.
- **📊 Interactive Data**: "Numbers that speak for themselves" with real-time counting animations.
- **📱 Mobile Optimized**: Full responsive design with custom cursor fallbacks and touch-friendly interactions.
- **🟢 Real Images**: High-quality Unsplash integration with robust error handling and fallbacks.

## 🛠️ Technology Stack

- **Frontend**: React 18, Vite
- **Animations**: GSAP (GreenSock Animation Platform)
- **Scrolling**: Lenis (Smooth Scroll)
- **Styling**: Tailwind CSS, Vanilla CSS (Premium Tokens)
- **State Management**: Redux Toolkit
- **Icons**: Lucide React / SVG Path animations

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/parthsharma17prs/hostprsel.git
   ```
2. **Install dependencies**
   ```bash
   cd frontend && npm install
   ```
3. **Run development server**
   ```bash
   npm run dev
   ```
